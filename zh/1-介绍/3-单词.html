单词

字母、数字和标点用来组成单词

单词之间可以包含任意个空白字符, 也可以没有空白字符

空白字符用来分隔单词, 防止歧义, 进行排版

#include <litan.h>

int main() {

    // 试着找出下列代码中的所有单词
    float radius;
    float pi;

    pi = 3.14;

    put_string("input radius: ");
    radius = get_float();

    put_string("the area of given radius is: ");
    put_float(pi * radius * radius);

    return 0;
}

类型

简单的类型可以用一个单词表示

类型可以看作数据的集合

缩写 英文全称 释义
void 空类型
bool boolean 逻辑(布尔)类型
char character 字符类型
int integer 整数类型
float 浮点数类型
string 字符串类型

数据

简单的数据可以用一个单词表示

C语言中的每一个数据都对应着一个类型

字符数据 以单引号 ' 开头和结尾

字符串数据 以双引号 " 开头和结尾

整数数据 和 浮点数数据 以数字开头,

true 和 false 是 逻辑数据, 分别代表真和假

标识符

标识符 是一种单词, 以字母、数字和下划线 _ 组成, 且不能以数字开头

类型, 数据, 运算符, 语法词, 四类单词之中, 都有满足标识符语法的单词

变量指的是代表数据的标识符

每个标识符都需要被定义, 以确定这个标识符的分类

标识符可以由C语言本身、标准库、第三方库或我们自己定义

关键字是其中C语言本身定义了的标识符

同一个标识符不能多次定义, 如自定义标识符不能与关键字冲突

自定义 pi radius

区分单词

从组成方式上, 单词可以分为五组

标识符, 符号, 数字数据, 字符数据, 字符串数据

通过单词的组成方式, 可以区分出代码中所有单词

代码风格

通过使用空白字符, 可以将C语言代码适当排版

实际上C语言代码仍然是一行字符, 只是回车字符显示成了换行效果,

下列程序虽然排版不当, 但是仍然可以运行

#include <litan.h>

int main() {

    // 不推荐把许多语句写在一行的风格
    put_string("input radius: "); float radius; radius = get_float(); float pi = 3.14;

    // 不推荐把一个语句写在多行的风格
    put_string
    (
    "the area of given radius is: "
    )
    ;

    // 推荐一个语句占一行
    put_float(pi * radius * radius);

    return 0;
}
