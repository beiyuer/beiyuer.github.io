语句的作用

语句 作用
空语句 什么也不做
表达式语句 计算, 赋值, 输入输出
定义语句 明确变量的类型, 分配存储空间
初始化语句 明确变量的类型, 分配存储空间, 并且给变量赋值
结构语句 改变语句的执行顺序

一个初始化语句, 作用相当于一个定义语句和一个赋值表达式语句

结构语句的作用将在第二章介绍

语句块

{ ... }

{ <语句> }

{ <语句1> <语句2> ... }

语句块是一种语句, 所以需要语句的地方, 都可以使用语句块

执行语句块, 相当于依次执行花括号内所有语句

{} 也是一个语句, 相当于空语句

#include <litan.h>

int main() {

    {{{{{;{{}}
    ;;;;{{{{{;
    put_string
    ("Hello");
    ;}};;;}}}}
    };;;{{}};;
    ;}};;;;;;}

    return 0;
}
