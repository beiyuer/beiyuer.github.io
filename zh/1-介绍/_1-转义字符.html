转义字符

在一个字符串中, 一些字符, 比如回车符, 不能直接出现在字符串中。 另一些字符, 比如 " 可能引起歧义

为了在字符串中使用无法直接出现的字符或预防歧义, 反斜杠 \ 被用来组成转义字符

#include <litan.h>

int main() {

    // 这个语句在屏幕上显示了三行字符
    // 字符串里使用 `\n` 两个字符代表 `<回车符>`
    put_string("*********\n* Hello *\n*********");

    return 0;
}
#include <litan.h>

int main() {

    // 这个语句语法正确, 因为使用了 `\"`两个字符代表 `"`一个字符
    put_string("I want put \" inside a string !");

    return 0;
}
#include <litan.h>

int main() {

    // 这个语句有歧义, 代码有语法错误, 从代码着色可以看出歧义来源
    put_string("I want put " inside a string !");

    return 0;
}

下表介绍了，在一个字符串中可能引起歧义的字符对应的转义字符

原字符 转义字符
<回车符> \n
<制表符> \t
<退格符> \b
" \"
\ \\
