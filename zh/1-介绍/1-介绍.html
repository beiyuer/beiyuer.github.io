介绍

第一章将会依次介绍C语言中 字符, 单词, 表达式和语句 的概念

字符组成单词, 单词组成表达式和语句, 并列的语句组成一份完整代码

#include <litan.h>

int main() {

    put_string("Hello");

    return 0;
}

注释行和预处理行

注释行以两个斜杠字符 // 开始, 直到一行的末尾

预处理行以一个井号字符 # 开始, 直到一行的末尾

注释行 和 预处理行 不需太多关注

// 注释行用于解释程序的作用, 让阅读代码更方便, 而且不影响程序运行

// `#include <litan.h>` 是一种预处理行, 可以把文件 `litan.h` 复制到代码中
// `litan.h` 是一个文件, 内容为其他人的C语言代码
#include <litan.h>

int main() {

    // 这个语句被执行, 把字符串 Hello 显示到了屏幕上
    put_string("Hello");

    return 0;
}
