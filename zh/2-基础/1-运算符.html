运算符

#include <litan.h>

int main() {

    // 算数运算符
    1 + 1; // 2
    3 * 4; // 12
    14 / 3; // 4
    11 % 3; // 2

    // 比较运算符
    5 < 6; // true
    5 >= 6; // false
    5 == 6; // false
    5 != 6; // true

    // 逻辑运算符
    !true; // false
    !false; // true
    true && false; // false
    true || false; // true

    return 0;
}
算数运算符 + - * / %
比较运算符 == != > < >= <=
逻辑运算符 ! && ||
赋值运算符 =
取内容运算符 *
取地址运算符 &
下标运算符 []
函数调用运算符 (,)
点运算符 .

函数调用

函数调用运算符 ( , ) 用来构成复合表达式

函数调用表达式是使用了 ( , ) 的复合表达式, 其中头部第一个表达式的类型是函数类型

参数, 是出现在 () 内, 由逗号 , 分隔的一类表达式

模式 语法 例子
f() <表达式>() get_float() pause()
f(x) <表达式>(<参数>) succeed(x)
f(x, y) <表达式>(<参数>, <参数>) add(x, y)
