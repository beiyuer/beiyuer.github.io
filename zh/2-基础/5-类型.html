简单类型, 可以用一个类型单词表示

复合类型, 需要其他类型形成具体类型

指针 数组 结构体 函数

定义语句

这一节解释定义语句的语法 <类型> <表达式>;

定义语句 <类型> <变量>;, 明确了变量的类型

定义语句 <类型> <表达式>; 明确了表达式的类型, 通过推断, 可以明确表达式内变量的类型

#include <litan.h>

int main() {
    float pi;
    int a[10];
    int *p;
}

float pi; 明确了 pi 是浮点数

int a[10]; 明确了 a[...] 是一个整数, 所以 a 是一个内容为整数的数组

int *p; 明确了 p 的内容是一个整数, 所以 p 是一个指向整数的指针

int main() { ... } 是一个初始化语句的特殊形式, int main() = { ... }; 是其完整形式。 其 <类型> <表达式> 部分明确了 main() 是一个整数, 所以 main 是一个返回整数、不带参数的函数。 main 函数被 { ... } 赋予初始值

类型定义语句

typedef <类型> <表达式>;

类型定义语句的一部分类似定义语句 <类型> <表达式>;

<类型> <表达式>; 明确了某个标识符是变量及其类型

通过 typedef 关键字, 使得这个标识符成为了类型本身

typedef int bool;
int false = 0;
int true = 1;

typedef char *string;

int bool;, 明确了 bool 是一个 整数型变量

通过 typedef, typedef int bool; 明确了 bool 不再是一个整数型变量, 而是整数类型本身(一个别名)

char *string;, 明确了 *string 是字符型, string 是指向字符型的指针

通过typedef, typedef char *string; 明确了 string 不再是一个指向字符的指针变量, 而是指向字符的指针类型本身
