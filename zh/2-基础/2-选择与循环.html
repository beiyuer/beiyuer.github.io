if 语句

if <表达式> <语句>

如果表达式等于 true, 语句会被执行

如果表达式等于 false, 此 if 语句会被跳过

其中, 表达式必须是圆括号表达式, 防止歧义


#include <litan.h>

int main() {

    put_string("input an integer: ");
    int x = get_int();

    if (x < 0) x = -x;

    put_string("The absolute value of it is ");
    put_int(x);

    return 0;
}

所有的 零值 都会被当作 false

所有的 非零值 都会被当作 true

零值 非零值
false 0 '\0' true 1 'a' 

#include <litan.h>

int main() {

    if (1) put_string("`1` is regarded as true!\n");

    return 0;
}

if-else 语句

if <表达式> <语句> else <语句>

如果表达式等于 true, 第一个语句块会被执行

如果表达式等于 false, else 后的语句块会被执行

#include <litan.h>

int main() {

    put_string("Welcome to Litan\'s Secret Database.\n");
    put_string("Password:");
    int password = get_int();

    if (password == 2333)
        put_string("Access Granted");
    else
        put_string("Access Denied");

    return 0;
}

while 语句

while <表达式> <语句>

如果表达式等于 true, 语句会被执行, 然后再次执行此 while 语句

如果表达式等于 false, 此 while 语句会被跳过

#include <litan.h>

int main() {

    int i;

    i = 1;
    while (i <= 10) {

        //////////////////////////////////////////////////
        // 在这一部分的代码, 将会执行10次, 其中i等于执行的次数
        //////////////////////////////////////////////////

        i = i + 1;
    }

    return 0;
}

for 语句

for (表达式1; 表达式2; 表达式3) <语句>

for语句 可以看作 while 语句的简写

表达式1; while 表达式2 { <语句> 表达式3; }

#include <litan.h>

int main() {

    int i;

    for (i = 1; i <= 10; i = i + 1) {
        //////////////////////////////////////////////////
        // 在这一部分的代码, 将会执行10次, 其中i等于执行的次数
        //////////////////////////////////////////////////
    }

    return 0;
}

代码风格

推论, 关键字, 表达式和语句间, 可以任意使用空白字符进行排版

  • if <表达式> 或 else 独占一行
  • 总是使用语句块
  • 语句块内的所有语句, 使用制表符进行缩写
  • 每在一层语句块内, 增加制表符进行缩进

语法正确, 风格不推荐

if <表达式> <语句> else <语句>
if <表达式> <语句>

推荐风格

if <表达式> 
{ 
    <语句1>
    <语句2>
    ...
}
if <表达式> 
{ 
    <语句1>
    <语句2>
    ...
}
else <表达式> 
{ 
    <语句3>
    <语句4>
    ...
}

嵌套的 if-else 语句, 推荐else后的语句不构成语句块, 进行特殊排版
