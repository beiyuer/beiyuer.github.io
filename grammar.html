计算机语言的一些概念的明确定义

词法部分

字母

字母的使用选择是人为的

例： 0

字母表alphabet，是字母的集合

例： { 0 , 1 }

单词/词元 token

单词是符号的可重复排列

例： [0, 0, 1] 可简写为 001

词典dictionary，是单词的集合

例： { 0 1 00 01 10 11 001 }

句子 sentence

句子是单词的可重复排列

例： [ 01 1010 0 0 0110 10 ]

语言 language

语言是句子的集合

例： { [0, 00], [101], [1, 1, 0] }

语系

语系是语言的集合

例：正规语系，正规语系是所有正规语言的集合

例：上下文无关语系，上下文无关语系是所有上下文无关语言的集合

例：上下文有关语系，

例：无限制语系，

四大形式语系（下面介绍）： 这四大形式语系形成了集合的全序真子集关系。即 a ⊂ b ⊂ c ⊂ d

字母、单词、句子、语言、语系等概念的意义和应用

五个概念的抽象都是有意义的

所有的数学问题、算法问题，都可以和句子是否属于语言的判断问题相互转换。即数学问题等价于形式语言的判断问题。

语系，对应了问题的难度

语言，对应了问题的表达方式。改变问题的表达方式不会显著降低问题难度

句子，对应一个问题的解决方法

单词，对应问题里一个涉及到的概念

符号，符号是有限的，但可以排列成无限个单词。

语系：四大语系对应了四大语法系统。一种语系对应了一类数学问题和对应解决方法，不同语系的问题的常用解法不同。
例如，正规语系可以用于字符串匹配问题，用于进行文件和数据库的增删改查的日常工作。
例如，上下文无关语系包含了所有的编程语言，用于创造程序。不同编程语言的编程方法类似。
例如，数学语言属于无限制语系，是一种无限制语言，很多数学问题不存在一种通用的解法。

语言：每种语言都有自己对对应的语法和词典。只有给定确定的语法才能进行句子是否属于语言的判断。
例如，不同软件系统可能提供了不同复杂度的正规语言以方便用户，有的软件好用，有的软件难用。
例如，每种编程语言都属于上下文无关语系，不同编程语言有不同的风格，使用者经常产生纠纷和冲突。
例如，同样是逻辑代数这门课，离散数学数字逻辑C语言汇编语言老师却使用了4套符号各教了一遍。

句子：一个语言中的句子数目是无限的，而一个句子的长度是有限的，一个句子里的单词数量也是有限的。
每个问题的具体解法就是一个句子。句子也可以叫做文章、程序。
例如：一段程序就是一个句子，长度有限，可以在有限的时间内编写运行，并解决实际的问题。
不可能把一个语言的所有句子穷举出来用于解决一类问题。

单词：单词的数目是无限的，可以表示无限多的新概念。
现实生活中的概念的数目是无穷的，不可能为每个概念创建一个不同的符号，但是可以为每个概念创建一个不同的单词。

符号：单词有无数种，而计算机底层只有0和1，没有无数种状态。必须用有限的状态的排列形成数量无限的符号。

语法部分

形式语言理论 formal language theory

形式语法的四部分：终结符号 中间符号 起始符号 生成式

形式语法和形式语言可以一一对应

终结符号集合

例：所有英语单词构成的英语词典

例：所有数字和运算符构成的集合

中间符号集合

例：很多语法概念，“主语” “谓语” “宾语” “名词成分” “名词” “冠词”

例：很多语法概念，表达式 多项式 初等函数的表示

起始符号

起始符号只有一个，无论叫什么。比如句子、文章、命题、表达式、多项式

起始符号，中间符号，终结符号。三种符号不应该有交集

生成式

生成式是（两个（符号的排列）），前一个排列不能是空排列 v

生成式是（（两个符号）的排列），前一个排列不能是空排列 x

例

“句子” => “主语” “谓语”

“句子” => “主语” “谓语” “宾语”

“谓语” => “谓语动词”

“谓语动词” => is

“主语” => this

“宾语” => “名词成分”

“名词成分” => “冠词” “名词”

“冠词” => a

“名词” => sentenece

上下文无关语言：大多数编程语言
